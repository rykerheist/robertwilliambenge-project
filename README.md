# Robert William Benge — E-Commerce Store

A full-stack e-commerce project built with [Medusa](https://medusajs.com/) (backend) and [Next.js](https://nextjs.org/) (storefront).

## Project Structure

```
robertwilliambenge-project/
├── backend/      # Medusa v2 backend (API, plugins, database)
└── storefront/   # Next.js storefront (Medusa Next.js Starter)
```

## Getting Started

### Backend

```bash
cd backend
npm install
npm run dev
```

The Medusa backend runs on `http://localhost:9000` by default.

### Storefront

```bash
cd storefront
npm install
npm run dev
```

The Next.js storefront runs on `http://localhost:8000` by default.

## Requirements

- Node.js 20+
- PostgreSQL
